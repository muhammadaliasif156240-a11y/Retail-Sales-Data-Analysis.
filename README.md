# Retail-Sales-Data-Analysis.
**Objective**: “To analyse retail transactions and identify highly profitable regions and products”.
**Tools Used**: 
A)	 Microsoft Excel (Data Cleaning),
B)	 Microsoft Power BI (Data Visualization and Data Analysis Expression),
C)	 MySQL (Data Querying).
Key Insights:
1)	East Region is leading the Revenue.
2)	Category Driving the most profit is “Fashion”.
3)	Total Orders peaked in the month of “December’.
4)	The Monthly Revenue Trend is Bullish.
